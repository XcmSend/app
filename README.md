## Build and run me:    
```shell
$ git clone git@github.com:XcmSend/app.git  
$ npm install -f
$ npm run build -f 
$ npm run dev
```

#### Build tailwind:

```
npx tailwindcss -c ./tailwind.config.js -i ./src/index.css -o dist/output.css
```
